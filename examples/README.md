# examples — reference builds

What coding agents *produced* from [`../ESSENCE.md`](../ESSENCE.md) in a [`../contexts/`](../contexts/)
environment — the *after* to the contexts' *before*. Kept out of `contexts/` so they never spoil the
exercise: build it yourself in a context first, then come here to see what a finished build looks like.

- [`fastapi-watchdog/`](fastapi-watchdog/) — a complete deployable watchdog **service** a fresh Claude
  agent built from the watchdog-agnostic `fastapi` context, including the `DECISIONS.md` it wrote.
